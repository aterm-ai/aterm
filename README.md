##

### Why Aterm?

- **Modern workspace**, built for agentic workflows.
- **Purpose-built panes** for working with agents.
- **Organized interface** that helps you focus and get the work done.

### Features

- **Windows, Pods, and Tabs**
Flexible environments for shifting development workspaces.
<img src="/assets/reattach-pod.png" width="600"/>
<img src="/assets/organize-tab.png" width="600"/>

- **Open Recent Project**
Opening a recent project launches all its terminals in the project's root directory.
<img src="/assets/recent-projects.png" width="600"/>

- **Dark / Light Mode**
Switch between themes to match your environment.
<img src="/assets/dark-mode.png" width="600"/>
